getUser(username) → {Object}

Sends the request to the server API for getting user profile object for the specified username. In case there was an unauthorized request attempt (meaning the user is not logged on the server side) the function and currently shown page is a FITeagle main page, the function triggers the sing out procedure, it is done for security reasons and other use cases. If the response is a successful one the function sets the received user as a current FITeagle clien user. In case the error code is responded the information about the event is written to the browser's console.
Parameters:
Name Type Description
username String to get the user profile object for.
